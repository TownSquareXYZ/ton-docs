# Огляд

Fift - це стекова мова програмування загального призначення, оптимізована для створення, налагодження та управління смарт-контрактами TON Blockchain.
Fift була спеціально розроблена для взаємодії з віртуальною машиною TON (TON VM або TVM) та блокчейном TON.

```fift
{ ."hello " } execute ."world"
hello world ok
```

:::info
Зазвичай використання Fift не є обов'язковим для програмування смарт-контрактів в TON. Однак, іноді вам може знадобитися використання мови Fift для вирішення нестандартних технічних проблем в рамках вашого завдання.
:::
