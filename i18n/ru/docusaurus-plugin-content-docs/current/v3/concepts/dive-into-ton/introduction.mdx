# The Open Network

**The Open Network (TON)** — это децентрализованная и открытая интернет-платформа, состоящая из нескольких компонентов. К ним относятся TON Blockchain, TON DNS, TON Storage, TON Sites и TON Proxy. Блокчейн TON — это основной протокол, соединяющий базовую инфраструктуру TON для формирования большей экосистемы TON.

TON нацелен на обеспечение повсеместной межсетевой совместимости при работе в высокомасштабируемой и безопасной среде. TON разработан для обработки миллионов транзакций в секунду с целью в конечном итоге охватить сотни миллионов пользователей в будущем.

_Блокчейн _TON разработан как распределенный суперкомпьютер, или “_суперсервер_”, предназначенный для предоставления разнообразных продуктов и услуг, способствующих развитию децентрализованного видения нового Интернета.
