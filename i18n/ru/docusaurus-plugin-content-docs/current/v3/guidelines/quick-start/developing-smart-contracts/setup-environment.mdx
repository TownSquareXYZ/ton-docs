# Настройка среды разработки

> **Вкратце:** В предыдущих шагах мы изучили концепцию смарт-контракта и основные способы взаимодействия с блокчейном TON с помощью **приложений кошельков** и **эксплореров**.

Это руководство описывает основные шаги настройки вашей среды разработки смарт-контрактов с помощью **`Blueprint SDK`** и создания базового шаблона проекта.

## Что требуется для прохождения

- Базовые навыки программирования.
- Знакомство с командной строкой.

## Настройка среды разработки

В этом руководстве мы используем [Blueprint SDK](https://github.com/ton-org/blueprint/) и стек [Node.js](https://nodejs.org/en)/TypeScript для написания обёрток, тестов и скриптов развёртывания вашего смарт-контракта, так как это обеспечивает самую лёгкую и готовую к использованию среду для разработки смарт-контрактов.

:::info

Использование нативных инструментов и языко-специфичных SDK для разработки смарт-контрактов рассмотрено в разделах документации более продвинутого уровня:

- [Компиляция и сборка смарт-контрактов в TON](/ru/v3/documentation/archive/compile#ton-compiler/).
- [Создание исходного состояния для развёртывания](/ru/v3/guidelines/smart-contracts/howto/wallet#creating-the-state-init-for-deployment/).
  :::

### Шаг 1: установите Node.js

Сначала посетите [страницу установки](https://nodejs.org/en/download) и выполните команды загрузки в `PowerShell` или `Bash`, в зависимости вашей операционной системе (Windows/Linux).

Проверьте, что `npm` и `node` установлены, с помощью следующей команды:

```bash
npm -v
node -v
```

### Шаг 2: выберите язык разработки смарт-контракта

В этом руководстве мы приводим примеры на языках `FunC`, `Tolk` и `Tact`. Вы можете выбрать любой из них и даже комбинировать смарт-контракты на разных языках. Чтобы воспользоваться руководством, не требуется глубокое понимание выбранного языка: достаточно базовых навыков программирования.

:::info
Краткий обзор языков можно найти здесь: [Языки программирования](/ru/v3/documentation/smart-contracts/overview#%D1%8F%D0%B7%D1%8B%D0%BA%D0%B8-%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F)
:::

### Шаг 3: установка Blueprint SDK

Измените директорию на родительскую папку вашего будущего проекта и выполните следующую команду:

```bash
npm create ton@latest
```

Это запустит интерактивный скрипт для создания шаблона проекта. Вы можете ввести всё, что хотите, но если вы хотите следовать тем же путям, что и в этом руководстве, выберите следующее:

1. **Имя проекта**: `Example`  
2. **Название первого контракта**: `HelloWorld`  
3. **Шаблон проекта**: **simple counter contract** на `FunC`, `Tolk` или `Tact`

Наконец, измените ваш текущий каталог на созданную папку с шаблоном проекта и установите все необходимые зависимости:

```bash
cd ./Example
npm install
```

### Шаг 4 (опционально): поддержка IDE и редактора

Сообщество TON разработало плагины, обеспечивающие поддержку синтаксиса в нескольких IDE и редакторах кода. Вы можете найти их здесь: [Список плагинов](https://docs.ton.org/ru/v3/documentation/smart-contracts/getting-started/ide-plugins/).

Также рассмотрите возможность установки плагинов, поддерживающих инструменты **JavaScript/TypeScript** для ваших любимых IDE или редактора кода, в частности, `Jest` для отладки тестов смарт-контрактов.

## Выберите язык программирования

Теперь, когда ваша среда настроена, выберите язык программирования, чтобы начать:

- **FunC или Tolk**: предполагаемое время обучения 25 минут для каждого.
- **Татья**: предполагаемое время обучения 15 мин.
