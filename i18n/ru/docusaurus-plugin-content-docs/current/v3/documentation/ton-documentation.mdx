## Разработка DApp

Децентрализованные приложения (DApps) - это приложения, которые запускаются в одноранговой сети компьютеров, а не на одном компьютере (блокчейн TON). Они похожи на традиционные веб-приложения, но построены поверх блокчейн сети. Это означает, что приложения децентрализованы, то есть ни одна организация не контролирует их.
