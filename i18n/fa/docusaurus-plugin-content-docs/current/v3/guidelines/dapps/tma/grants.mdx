# کمک‌هزینه‌ها

## کمک‌هزینه‌های Web3 تلگرام

برای تقویت بیشتر نوآوری، [بنیاد TON](https://ton.foundation/en) برنامه [کمک‌هزینه‌های Web3 تلگرام](http://t.me/toncoin/991) را راه‌اندازی کرده است. این ابتکار برای تشویق بیشتر توسعه‌دهندگان به ایجاد پلتفرم‌های جدید یا انتقال پلتفرم‌های موجود به TON و تلگرام طراحی شده است.
