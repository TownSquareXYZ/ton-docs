# سایت‌های TON برای برنامه‌ها

## چگونه سایت TON را در برنامه خود پشتیبانی کنید؟

شما می‌توانید با یکپارچه‌سازی یک پراکسی ورود محلی در محصول خود، سایت‌های TON را از برنامه خود راه‌اندازی کنید.

یک کتابخانه و مثال‌هایی در زیر موجود است:

- [Andriod](https://github.com/andreypfau/tonutils-proxy-android-example)
- [iOS](https://github.com/ton-blockchain/ton-proxy-swift)

:::caution
پراکسی‌های ورود عمومی فقط برای آشنایی و آزمایش راه‌اندازی شده‌اند؛ لطفاً از آن‌ها استفاده نکنید.
:::
