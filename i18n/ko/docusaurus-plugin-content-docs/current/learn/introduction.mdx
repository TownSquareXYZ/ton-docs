# 개방형 네트워크

오픈 네트워크(TON)\_\_는 여러 구성 요소로 이루어진 탈중앙화된 개방형 인터넷 플랫폼입니다. 여기에는 다음이 포함됩니다: TON 블록체인, TON DNS, TON 스토리지, TON 사이트. TON 블록체인은 TON의 기본 인프라를 하나로 연결하여 더 큰 TON 생태계를 형성하는 핵심 프로토콜입니다.

TON은 확장성이 뛰어난 보안 프레임워크에서 운영되는 동시에 광범위한 크로스체인 상호운용성을 달성하는 데 중점을 두고 있습니다. TON은 초당 수백만 건의 트랜잭션을 처리할 수 있도록 설계되었으며, 향후 수억 명의 사용자에게 도달하는 것을 목표로 하고 있습니다.

톤 블록체인은 분산형 슈퍼컴퓨터, 즉 '_슈퍼서버_'로 설계되어 새로운 인터넷의 탈중앙화 비전 개발에 기여하기 위한 다양한 제품과 서비스를 제공합니다.
