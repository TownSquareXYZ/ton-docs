# MyTonCtrl 상태

`status` 명령어 출력에 대한 설명입니다.

![status](/img/docs/mytonctrl/status.png)

## TON 네트워크 상태

### 네트워크 이름

가능한 값: `mainnet`, `testnet`, `unknown`. `unknown`은 절대 출력되면 안 됩니다.

### 검증자 수

두 개의 값이 있습니다: 녹색과 노란색. 녹색은 온라인 검증자 수, 노란색은 전체 검증자 수입니다.
0보다 큰 정수여야 하며, MyTonCtrl은 `getconfig 34` 명령으로 이를 가져옵니다.
자세한 내용은 [이 섹션(매개변수 32-34-36)](/v3/documentation/network/configs/blockchain-configs#param-32-34-and-36)을 확인하세요.

### 샤드체인 수

0보다 큰 정수여야 하며, 녹색으로 표시됩니다.

### 제안 수

두 개의 값이 있습니다: 녹색과 노란색. 녹색은 `새 제안` 수, 노란색은 `전체 제안` 수입니다.

### 불만사항 수

두 개의 값이 있습니다: 녹색과 노란색. 녹색은 `새 불만사항` 수, 노란색은 `전체 불만사항` 수입니다.

### 선거 상태

녹색 텍스트 `open` 또는 노란색 `closed`일 수 있습니다.

## 로컬 검증자 상태

### 검증자 인덱스

검증자 인덱스가 0 이상인 경우(검증자 모드가 활성화된 경우에는 그래야 함) 녹색, 그렇지 않으면 빨간색입니다.

### 로컬 검증자의 ADNL 주소

단순 ADNL 주소입니다.

### 로컬 검증자 지갑 주소

스테이킹에 사용되는 주소는 유효한 TON 주소여야 합니다.

### 로컬 검증자 지갑 잔액

지갑의 잔액입니다.

### 로드 평균

`[int]: int, int, int` 형식입니다. 첫 번째는 CPU 수, 나머지는 지난 1, 5, 15분 동안의 시스템 로드 평균을 나타냅니다.

### 네트워크 로드 평균

세 개의 정수로, `로드 평균`과 동일한 논리: 지난 1, 5, 15분 동안의 시스템 로드 평균입니다.

### 메모리 로드

RAM과 스왑의 절대적, 상대적 메모리 사용량을 나타내는 두 쌍의 정수입니다.

### 디스크 로드 평균

모든 디스크에 대해 `메모리 로드`와 동일합니다.

### Mytoncore 상태

녹색이어야 하며, Mytoncore가 가동된 시간을 알려줍니다.

### 로컬 검증자 상태

녹색이어야 하며, 로컬 검증자가 가동된 시간을 알려줍니다.

### 로컬 검증자 동기화 해제

정수는 20 미만이어야 합니다(그러면 녹색으로 표시됩니다).

### 로컬 검증자 마지막 상태 직렬화

서비스가 중단된 마스터체인 블록의 수를 보여줍니다.

### 로컬 검증자 데이터베이스 크기

절대 로드는 1000 GB 미만이어야 하고, 상대 로드는 80% 미만이어야 합니다.

### mytonctrl 버전

커밋의 해시와 브랜치 이름입니다.

### 검증자 버전

커밋의 해시와 브랜치 이름입니다.

## TON 네트워크 설정

### 설정자 주소

설정자 주소, 자세한 내용은 [이 매개변수 0](/v3/documentation/network/configs/blockchain-configs#param-0)을 확인하세요.

### 선거인 주소

선거인 주소, 자세한 내용은 [이 매개변수 1](/v3/documentation/network/configs/blockchain-configs#param-1)을 확인하세요.

### 검증 기간

초 단위의 검증 기간, 자세한 내용은 [이 매개변수 15](/v3/documentation/network/configs/blockchain-configs#param-15)를 확인하세요.

### 선거 기간

초 단위의 선거 기간, 자세한 내용은 [이 매개변수 15](/v3/documentation/network/configs/blockchain-configs#param-15)를 확인하세요.

### 보류 기간

초 단위의 보류 기간, 자세한 내용은 [이 매개변수 15](/v3/documentation/network/configs/blockchain-configs#param-15)를 확인하세요.

### 최소 스테이크

TON 단위의 최소 스테이크, 자세한 내용은 [이 매개변수 17](/v3/documentation/network/configs/blockchain-configs#param-17)을 확인하세요.

### 최대 스테이크

TON 단위의 최대 스테이크, 자세한 내용은 [이 매개변수 17](/v3/documentation/network/configs/blockchain-configs#param-17)을 확인하세요.

## TON 타임스탬프

### TON 네트워크 출시

현재 네트워크(메인넷 또는 테스트넷)가 출시된 시간입니다.

### 검증 주기 시작

검증 주기의 시작 타임스탬프; 미래 시점을 나타내는 경우 녹색으로 표시됩니다.

### 검증 주기 종료

검증 주기의 종료 타임스탬프; 미래 시점을 나타내는 경우 녹색으로 표시됩니다.

### 선거 시작

선거의 시작 타임스탬프; 미래 시점을 나타내는 경우 녹색으로 표시됩니다.

### 선거 종료

선거의 종료 타임스탬프; 미래 시점을 나타내는 경우 녹색으로 표시됩니다.

### 다음 선거 시작

다음 선거의 시작 타임스탬프; 미래 시점을 나타내는 경우 녹색으로 표시됩니다.
