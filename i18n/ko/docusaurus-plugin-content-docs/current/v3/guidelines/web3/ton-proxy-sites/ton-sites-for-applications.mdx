# 애플리케이션용 TON 사이트

## 애플리케이션에서 TON 사이트를 지원하는 방법

제품에 로컬 진입-프록시를 통합하여 애플리케이션에서 TON 사이트를 실행할 수 있습니다.

기존 라이브러리와 다음 예제들이 있습니다:

- [안드로이드](https://github.com/andreypfau/tonutils-proxy-android-example)
- [iOS](https://github.com/ton-blockchain/ton-proxy-swift)

:::caution
공개 진입-프록시는 숙지와 테스트 목적으로만 출시되었으니 사용하지 마세요.
:::
