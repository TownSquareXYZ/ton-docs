# 시작하기

DApp 개발을 시작하기 전에 기본적인 블록체인 원리를 이해해야 합니다. [The Open Network](/v3/concepts/dive-into-ton/introduction)와 [Blockchain of Blockchains](/v3/concepts/dive-into-ton/ton-blockchain/blockchain-of-blockchains) 문서가 도움이 될 것입니다.

TON DApp은 백엔드 없이 블록체인과 상호작용하는 애플리케이션입니다. 대부분의 경우 커스텀 [스마트 컨트랙트](/v3/documentation/smart-contracts/overview)와 상호작용합니다. 이 문서는 예시와 빠른 DApp 개발을 위해 TON에서 사용 가능한 표준 자산을 처리하는 방법을 제공합니다.

DApp은 일반적으로 TON용 SDK가 있는 모든 프로그래밍 언어로 작성할 수 있습니다. 실제로 가장 일반적인 선택은 웹사이트이며, 그 다음으로 텔레그램 미니 앱입니다.
