# TON 문서

### TON 코스

TON 블록체인에 대한 종합적인 가이드인 **TON 블록체인 과정**을 소개하게 되어 자랑스럽게 생각합니다. 이 과정은 TON 블록체인에서 스마트 컨트랙트와 탈중앙화 애플리케이션을 매력적인 대화형 방식으로 만드는 방법을 배우고자 하는 개발자를 위해 설계되었습니다.

9개의 모듈\_\_로 구성되어 있으며 TON 블록체인의 기본 사항, FunC 프로그래밍 언어, TON 가상 머신(TVM)을 다룹니다.

## DApp 개발

탈중앙화 애플리케이션(DApp)은 단일 컴퓨터(TON 블록체인)가 아닌 컴퓨터의 P2P 네트워크에서 실행되는 애플리케이션입니다. 기존 웹 애플리케이션과 유사하지만 블록체인 네트워크 위에 구축됩니다. 이는 디앱이 탈중앙화되어 있다는 것을 의미하며, 어떤 단일 주체가 디앱을 통제하지 않는다는 것을 의미합니다.
