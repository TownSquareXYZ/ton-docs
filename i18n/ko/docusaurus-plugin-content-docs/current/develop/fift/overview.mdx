# 개요

파이브는 TON 블록체인 스마트 컨트랙트 생성, 디버깅 및 관리에 최적화된 스택 기반 범용 프로그래밍 언어입니다.
Fift는 TON 가상머신(TON VM 또는 TVM) 및 TON 블록체인과 상호 작용하도록 특별히 설계되었습니다.

```fift
{ ."hello " } execute ."world"
hello world ok
```

:::info
일반적으로 TON에서 스마트 컨트랙트를 프로그래밍할 때 파이프를 사용할 필요는 없습니다. 그러나 때때로 작업의 일부로 흔하지 않은 기술적 문제를 해결하기 위해 파이프 언어를 사용해야 할 수도 있습니다.
:::
