# 토큰(FT, NFT)

[분산형 TON 토큰 개요](https://telegra.ph/Scalable-DeFi-in-TON-03-30)

> TON 기반 토큰과 NFT는 단일 센터가 없으며 병목 현상을 일으키지 않습니다.
>
> 주어진 컬렉션의 각 NFT는 별도의 스마트 컨트랙트입니다. 각 사용자의 토큰 잔액은 별도의 사용자 지갑에 저장됩니다.
>
> 스마트 컨트랙트는 서로 직접 상호 작용하여 전체 네트워크의 부하를 분산시킵니다.
>
> 사용자 및 트랜잭션 수가 증가해도 부하가 균등하게 유지되어 네트워크 확장이 가능합니다.

### 제튼 디플로이러

제톤은 TON 블록체인의 맞춤형 대체 가능한 토큰입니다. 아래의 제톤 디플로이러 예시를 사용하여 TON 블록체인에서 나만의 토큰을 생성할 수 있습니다:

- **[톤 민터](https://minter.ton.org/)** - 오픈 소스 Jetton 배포자 dApp
- [제튼 디플로이러 - 계약](https://github.com/ton-defi-org/jetton-deployer-contracts) (FunC, TL-B)
- [제튼 디플로이러 - 웹클라이언트](https://github.com/ton-defi-org/jetton-deployer-webclient) (React, 타입스크립트)

### Jettons와 함께 작업할 수 있는 도구

- [NFT 제톤 판매 계약](https://github.com/dvlkv/nft-jetton-sale-smc) - 제톤 지원과의 NFT 판매 계약
- [스케일톤](http://scaleton.io) - 사용자 지정 토큰 잔액 보기
- [@tegro/ton3-client](https://github.com/TegroTON/ton3-client#jettons-example)-SDK로 제톤에 대한 정보를 조회할 수 있습니다.
