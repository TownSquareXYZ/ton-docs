# TON Connect 2.0 대 1.0

톤 커넥트 2.0은 톤 커넥트 1.0에 존재했던 많은 문제를 해결합니다.

톤 커넥트 2.0 프로토콜은 최고 수준의 보안, 탈중앙화 애플리케이션(dApp) 개발을 위한 개발자 친화적인 환경, 실시간 UX로 구동되는 간소화된 사용자 경험을 제공합니다.
