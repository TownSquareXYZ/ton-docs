# 지갑 연결

지갑 개발자는 지갑을 TON 커넥트에 연결하여 사용자가 안전하고 편리한 방식으로 TON 앱과 상호 작용할 수 있도록 할 수 있습니다.

## 통합

다음 단계에 따라 지갑을 TON Connect에 연결하세요:

1. 프로토콜 사양](/개발/앱/톤커넥트/프로토콜/)을 자세히 읽어보세요.
2. SDK](/개발/앱/톤-커넥트/개발자) 중 하나를 사용하여 프로토콜을 구현합니다.
3. 풀 리퀘스트를 통해 지갑을 [지갑 목록](https://github.com/ton-blockchain/wallets-list)에 추가하세요.

```mdx-code-block
<Button href="/develop/dapps/ton-connect/protocol/" colorType={'primary'} sizeType={'sm'}>
```
