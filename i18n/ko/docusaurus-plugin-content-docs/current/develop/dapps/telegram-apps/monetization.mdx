# 수익 창출

## 월렛 결제

<br></br>
<img src="https://storage.googleapis.com/ton-strapi/reason_Card5_19eeac1401/reason_Card5_19eeac1401.png" alt="Wallet Pay illustration"/>
<br></br>
월렛 페이는 텔레그램 미니앱의 기본 결제 시스템으로, 암호화폐와 법정화폐 거래를 모두 지원합니다. 주문 통계를 모니터링하고 쉽게 자금을 인출할 수 있습니다.
월렛 생태계에 내장된 월렛 페이는 판매자와 고객간의 원활한 금융거래를 지원합니다.

유용한 링크:

- [월렛 페이 비즈니스 지원](https://t.me/WalletPay_supportbot)은 월렛 페이 지원팀에 문의할 수 있는 텔레그램 봇입니다.
- [데모 스토어 봇](https://t.me/PineAppleDemoWPStoreBot)은 월렛 페이 기능 소개를 위한 텔레그램 봇입니다. (주의: 모든 결제는 실제 자산으로 진행됩니다.)
- [판매자 커뮤니티](https://t.me/+6TReWBEyZxI5Njli)는 그룹 구성원 간에 경험과 솔루션을 공유하기 위한 텔레그램 그룹입니다.

톤 커넥트는 톤의 **지갑**과 **앱** 간의 통신 프로토콜입니다.

TON에 구축된 **앱**은 풍부한 기능과 고성능을 제공하며 스마트 컨트랙트를 통해 사용자 자금을 보호하도록 설계되었습니다. 앱은 블록체인과 같은 탈중앙화 기술을 사용하여 구축되기 때문에 일반적으로 탈중앙화 애플리케이션(dApp)이라고 불립니다.

**지갑**은 거래를 승인하는 UI를 제공하고 사용자의 암호화 키를 개인 디바이스에 안전하게 보관합니다. 이러한 분리는 사용자에게 빠른 혁신과 높은 수준의 보안을 제공합니다. 지갑은 자체적으로 폐쇄적인 생태계를 구축할 필요가 없고, 앱은 최종 사용자의 계정을 보관하는 위험을 감수할 필요가 없습니다.

TON Connect는 지갑과 앱 간에 원활한 사용자 경험을 제공하는 것을 목표로 합니다.

```mdx-code-block

```

## TON 구독

TON 블록체인의 거래는 빠르고 네트워크 수수료가 낮기 때문에 스마트 컨트랙트를 통해 온체인에서 반복 결제를 처리할 수 있습니다.

예를 들어, 사용자는 디지털 콘텐츠(또는 다른 모든 콘텐츠)를 구독하고 월 1톤의 요금이 청구될 수 있습니다.
