# 시작하기

디앱에 대해 알아보기 전에 기본 블록체인의 원리를 이해해야 합니다. 오픈 네트워크](/학습/소개) 및 [블록체인의 블록체인](/학습/개요/톤블록체인) 문서가 도움이 될 수 있습니다.

TON 디앱은 블록체인과 상호작용하는 백엔드가 없는 애플리케이션입니다. 대부분의 경우 커스텀 [스마트 컨트랙트](/개발/스마트 컨트랙트/)와 상호작용하며, 이 문서는 예시로서 그리고 더 빠른 디앱 개발을 위해 TON에서 사용할 수 있는 표준 자산을 처리하는 방법을 제공합니다.

디앱은 일반적으로 TON용 SDK가 있는 모든 프로그래밍 언어로 작성할 수 있습니다. 실제로 가장 많이 선택되는 것은 웹사이트이며, 그 다음이 텔레그램 미니앱입니다.

```mdx-code-block
<Button href="/develop/dapps/telegram-apps/" colorType={'primary'} sizeType={'sm'}>
```

TMA 구축

```mdx-code-block
</Button>
```

```mdx-code-block
<Button href="/develop/dapps/apis/sdk" colorType="secondary" sizeType={'sm'}>
```
