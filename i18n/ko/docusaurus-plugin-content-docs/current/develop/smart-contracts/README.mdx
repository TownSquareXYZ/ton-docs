# 소개

TON 블록체인의 스마트 컨트랙트 생성, 개발 및 배포는 [FunC 프로그래밍 언어](/개발/스마트컨트랙트/#func-language)와 [TON 가상 머신(TVM)](/개발/스마트컨트랙트/#ton-virtual-machine)을 활용합니다.

### TON 코스

톤 블록체인에 대한 종합적인 가이드인 **톤 블록체인 코스**를 소개하게 되어 자랑스럽게 생각합니다. 이 과정은 TON 블록체인에서 스마트 컨트랙트와 탈중앙화 애플리케이션을 만드는 방법을 배우고자 하는 개발자를 위해 설계되었습니다.

## 스마트 컨트랙트의 예

TON 커뮤니티에서 제공하는 기성 스마트 컨트랙트 예시와 도구를 살펴보세요.

:::info 작은 팁
FunC\_를 사용하여 작성된 스마트 컨트랙트에 집중하세요. 하위 언어인 파이브(_.fif) 대신 펀씨(_.fc)를 사용해 작성된 스마트 컨트랙트에 집중하는 것이 더 나은 경우가 많습니다.
:::

TON에서 스마트 컨트랙트의 표준 예시에는 지갑, 선거인(TON에서 검증을 관리하는), 다중 서명 지갑이 있으며, 이는 공부할 때 참고할 수 있습니다.

## 커뮤니티 도구

- [disintar/toncli](/개발/스마트-컨트랙트/sdk/toncli) - toncli는 FunC 컨트랙트를 빌드, 배포 및 테스트하는 데 사용되는 명령줄 인터페이스입니다.

- [tonwhales.com/tools/boc](https://tonwhales.com/tools/boc) - BOC 파서
- [tonwhales.com/tools/introspection-id](https://tonwhales.com/tools/introspection-id) - crc32 생성기
