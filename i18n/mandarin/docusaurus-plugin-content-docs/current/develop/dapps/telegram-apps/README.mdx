Telegram Mini App (or TMAs) 是在 Telegram messenger 中运行的网络应用程序。 它们是使用网页技术——HTML、CSS 和 JavaScript来构建的。

:::tip
既然TMA是网页并使用 JavaScript，您需要选择 [JS/TS 基于 SDK](/develop/dapps/apis/sdk#typescript--javascript)。
:::

## 概览

Telegram bot 可以完全替换任何网站。 他们支持通过20个付款提供商进行无缝的授权和集成付款（其中包括谷歌付款和苹果付款）。 向用户发送定制推送通知等等。

使用 Mini App，机器人可以获得一个全新的维度。 机器人开发者可以使用世界上最广泛使用的编程语言JavaScript创建无限灵活的接口。

这里有一些关于 Telegram Mini App的关键点：

- **Integration in Telegram** : Telegram Mini Apps 是为了无缝地整合到 Telegram app, 为用户提供一个有凝聚力的体验。 他们可以在 Telegram 聊天或群組聊天中访问。
- **增强功能**: Telegram Mini Apps 可以提供广泛的功能。 它们可用于各种目的，如游戏、内容分享、生产率工具等等。 这些应用将 Telegram 平台的功能扩展到基本消息之外。
- **跨平台兼容性**：既然Telegram Mini Apps 是基于网络的，它们可以在 Android、iOS、PC、Mac 和 Linux Telegram 应用上访问。 用户可以访问它们，而无需在一次单击中添加安装。
- **Bot Interaction**: Telegram Mini Apps 经常使用 Telegram bots 来提供交互和自动化体验。 机器人可以响应用户输入，执行任务，并在Mini App内促进交互。
- **开发框架**：开发者可以使用 HTML、CSS 和 JavaScript 等网页开发技术构建Telegram Mini 应用程序。 此外，Telegram 提供了开发者工具和 API，用于创建这些应用并将其整合到 Telegram 平台。
- **货币化机会**: Telegram Mini Apps 可以通过各种方式货币化，例如通过应用内购买， 订阅模式或广告，使其对开发商和企业具有吸引力。
- **Web3 就绪**: TON SDK; TON Connect 是钱包和TON应用程序之间的通信协议; 令牌
- **Community Development**：Telegram 有一个蓬勃发展的开发者社区，许多第三方开发者创建他们的 Telegram Mini 应用程序并与用户分享。 这种由社区驱动的做法促进了现有应用的创新和多样性。

总之，Telegram Mini Apps 提供额外的功能和服务，作为增强Telegram 经验的一种手段。 同时为开发者提供在Telegram 生态系统中创建和分配其应用程序的机会。
