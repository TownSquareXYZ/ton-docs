# 小贴士和砖块

在这个页面上，你会找到一个常见问题的列表，这些问题与TMA中的问题有关。

### 如何解决TMA中的缓存溢出问题？

:::tip
只有重新安装 Telegram 应用程序才会有帮助。
:::

### 是否有关于缓存HTML文件头的建议？

:::tip
It's preferable to switch off cache in the HTML. To ensure your cache switched off, specify headers in your request according the following:

```curl
Cache-Control: no-store, must-revalidate
Pragma: no-cache
Expires: 0
```

:::

### 发展信息交流中心建议采取哪些措施促进发展？

由于熟悉的开发工具，谷歌Chrome的开发进程更加方便。

您可以检索微型应用程序的启动参数，并在 Chrome 中打开此链接。 就我们而言，最简单的方法是从 web 版本的 Telegram 获取启动参数：<https://web.telegram.org/>

### 关闭行为

在许多网页应用程序中，用户可能会在滚动到顶部时意外关闭应用程序。 如果他们拖动应用程序的部分太远，无意中触发应用程序关闭，就会发生这种情况。

<p align="center">
    <br />
    <img width="240" src="/img/docs/telegram-apps/closing-behaviour.svg" alt="closing_behaviour_durgerking" />
    <br />
</p>

为了防止这种意外关闭，启用 "closing_behavior" 在 TMA 中。 此方法将添加一个对话框，用户可以在哪里批准或拒绝关闭Web 应用程序。

```typescript
window Telegram.WebApp.enable. 关闭()
```
