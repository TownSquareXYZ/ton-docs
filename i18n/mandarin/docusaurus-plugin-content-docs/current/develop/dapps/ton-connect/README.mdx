## 鸟的眼镜

**应用程序** 建立在TON上提供了丰富的功能和高性能，旨在通过智能合约保护用户资金。 由于应用程序是使用分散的技术，例如Blockchain来构建的，它们通常被称为分散的应用程序 (dApp)。

**钱包** 提供用户界面批准交易，并在个人设备上安全地保持用户的加密密钥。
这种区分问题使得用户能够迅速创新和高度安全：钱包不需要建造网上花园生态系统。 而应用程序不需要使用风险控件的终端用户账户。

TON Connect 旨在提供钱包和应用之间无缝的用户体验。
