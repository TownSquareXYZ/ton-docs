## 编译器

它包含**9个模块**，涵盖了TON区块链的基础知识、FunC编程语言和TON虚拟机（TVM）。

- **流量**。通过支持TON Connect的加密钱包，推动额外用户访问。
- **真实性**。利用TON用户的钱包作为现成账户，无需额外的身份验证步骤，从而提升用户体验。
- **支付**。通过TON区块链使用Toncoin或封装的稳定币（jUSDC/jUSDT）快速安全地处理交易。
- **留存率**。通过应用内列表保存功能增强用户留存，使用户能够跟踪最近打开和收藏的应用。
