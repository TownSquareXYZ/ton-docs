## 为您的 DApp 使用客户反馈

探索TON生态系统提供的这些可达到的目标

- **贩运** 通过支持TON连接的加密钱包驱动更多用户访问。
- **认证**。 利用TON用户的钱包作为现成帐户，消除额外认证步骤的需要，从而增加用户体验。
- **付款** 使用Tonco币或包装稳定硬币通过TON区块链快速和安全地处理交易(jUSDC/jUSDT)。
- **保留**。 通过应用程序内存列表保存功能加强用户保留，这种功能允许用户跟踪最近打开和最喜欢的应用程序。

## 加入Ton星系

要将您的服务与 TON 生态系统连接起来，您需要实现以下内容：

- **TON Connect**. 将TON Connect 协议纳入您的应用中。
- **交易**。 使用 TON 库创建指定的交易消息。 通过我们的全面指南深入到[发送消息](/develop/dapps/ton-connect/message-builders)的过程中。
- **付款** 通过公共API进行付款 ([tonapi](https://tonapi.io/)) 或您自己的索引，例如 [gobycicle](http://github.com/gobicycle/bicycle)。 了解更多关于[资产处理](/develop/dapps/asset-processing)的广泛指南。
