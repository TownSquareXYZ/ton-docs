# TON 连接 2.0 vs 1.0

TON Connect 2.0 解决了许多存在于TON Connect 1.0中的问题。

TON Connect 2。 协议为开发分散的应用程序提供了最高级别的安全环境(dApp)， 有利于开发者的环境。 和由实时UX驱动的简化用户体验。
