# 正在开始

在潜入DApp之前，请确保您了解底层区块链原则。 您可能会发现我们的 [开放网络] (/learn/introduction) 和 [Blockchain 的 Blockchain] (/learn/overviews/ton-blockchain) 文章是有用的。

TON DApps 是没有后端的应用，它与区块链交互。 在大多数情况下，它们与习惯[智能合同](/develop/smart-contracts/)相互作用。 此文档提供了处理TON中现有的标准资产的方法，既可以作为样板，也可以更快地开发DApp。

DApp 一般可以用任何为 TON使用 SDK 的编程语言写入。 实际上，最常见的选择是一个网站，然后是Telegram mici-app。
