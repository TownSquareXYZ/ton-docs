# 令牌 (FT, NFT)

[发行版TON tokens overview](https://telegra.php/Scalable-DeFiin-TON-03-30)

> TON驱动令牌和 NFTs 没有一个中心，不会造成瓶颈。
>
> 给定收藏中的每个NFT都是单独的智能合同。 每个用户的代币余额存储在一个单独的用户钱包中。
>
> 智能合约直接相互作用，在整个网络上散布负荷。
>
> 随着用户和交易数的增长，负荷仍然是均衡的，使网络能够缩小。
