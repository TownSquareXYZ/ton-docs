[Hacktoberfest](https://hacktoberfest.digitalocean.com/) 是一个长达一个月的庆祝活动，它是 _开源项目_，他们的 \*_维护者_和整个社区_贡献者_。 每年10月，开源维护者在引导开发者第一次拉取请求时给予新贡献者更多关注。

对于TON社区来说，现在是帮助生态系统共同发展的时候了。 所以让我们加入我们的 **Hack-TON-berfest** 组加入整个世界，成为_#1 年开源生态系统_！

## 如何参与？

2022年黑莓智能规则如下：

- **Hacktoberfest向每个人开放**！
- 注册9月26日至10月31日期间的任何时间
- 可以在 GITHUB 或 GITLAB项目中合并请求：
  - [TON生态系统项目清单](/hacktonberfest)
  - [在 GitHub 上的项目列表](https://github.com/topics/hacktoberfest)
- 有 **4** 10月1日至10月31日期间接受的拉/合并请求
- 前40岁， 00位完成Hacktoberfest的参与者(维护者和贡献者)可以选择两种奖品：一种树在他们的名誉上或一种Hacktoberfest 2022 t-shirt。 _(来自黑莓智能社区)_
- TON生态系统项目的每个参与者(维护者和贡献者)将收到一份[**有限的 Hack-TON-berfest NFT**](#what-the-rewards)。 _(来自 TON Foundation)_

对所有在TON的人来说，这是一个推动整个生态系统增长的机会，并从TON基金会获得很酷的奖励。 让我们一起去做吧！

## 奖励是什么？

为了激励社区为TON生态系统中的开放源码项目做出贡献，您将能够从TON基金会得到特别奖励。 每个参与者将收到**有限的 Hack-TON-berfest NFT** 成就作为参与证明：
